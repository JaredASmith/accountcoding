CSV to Pivot Table Converter

This tool processes CSV files to generate categorized pivot tables based on custom logic. It's ideal for accounting tasks, data analysis, and more!

How It Works

  1. Upload a CSV file with a column named Description and Amount.
  2. The tool categorizes data based on predefined text patterns like CDTFA and BANKCARD.
  3. It generates a pivot table summarizing amounts by category.

Sample Code

Below is a snippet of the logic used for categorizing the data:

def categorize_cell(value):
    if isinstance(value, str):
        if 'CDTFA' in value:
            return 'Sales Tax'
        elif 'BANKCARD' in value:
            return 'Deposit'
    return 'Other'
